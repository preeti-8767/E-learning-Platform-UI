Responsive Front-End E-Learning Platform featuring course listings, progress tracking, and video embedding with a styled and interactive multi-page interface built using HTML, CSS, and JavaScript.

# E-Learning Platform (Front-End)

A modern and responsive **Front-End E-Learning Platform** built using HTML, CSS, and JavaScript.  
This project demonstrates a styled and interactive multi-page interface with features like course listing, progress tracking, and video embedding.

# Features

- ✅ Multi-Page Interface
- ✅ Course Listing Section
- ✅ Individual Course Detail Pages
- ✅ Video Embedding for Lessons
- ✅ Progress Tracking System
- ✅ Interactive UI Components
- ✅ Responsive Design (Mobile Friendly)
- ✅ Clean and Modern User Interface

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Media Queries
